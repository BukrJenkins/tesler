Доступность возможности массовых изменений для List-виджетов контролируется флагом
 <p>
  {
      "bulkChangesAvailable": true
  },
  </p>
  который необходимо задать в поле OPTIONS.
  <br/>
  Переход в режим Массовых изменений осуществляется по нажатию кнопки в меню Дополнительно ("шестеренка")
  <br/>
  <b style="color: red;">ЗАПРЕЩЕНО ИСПОЛЬЗОВАНИЕ МАССОВЫХ ИЗМЕНЕНИЙ ДЛЯ ПОЛЕЙ С ТИПОМ "pickList"</b> ()
  <br/>
  Массовые изменения (МИ) доступны только для одного поля. После осуществления МИ для выбранного поля, возможен переход к следующему полю.
  <br/>
  <br/>
  Клиентская валидация осуществляется только для полей с типом Dictionary. Для всех остальных типов выбранное для МИ значение отправляется на сервер.
  Этот же механизм применяется и для выполнения кастомных действий (валидация на сервере). На клиенте происходит только создание списка всех доступных действий для всех строк.
