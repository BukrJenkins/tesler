<h3>В файл *.sqlbc.json добавляем для бк "editable": "1"</h3>
<br>
<h3>В файл *.widget.json добавляем на виджет поля из 3-х возможных типов (строка, целое число, дата):</h3>
<p>- edit_string1, edit_string2, ... (10 полей)</p>
<p>- edit_number1, edit_number2, ... (5 полей)</p>
<p>- edit_date1, edit_date2, ... (5 полей)</p>
<p>- edit_lov1, edit_lov2, ... (5 полей)</p>
<p> Значения для полей редактируемых справочников задаются через администрированние связных справочников</p>
