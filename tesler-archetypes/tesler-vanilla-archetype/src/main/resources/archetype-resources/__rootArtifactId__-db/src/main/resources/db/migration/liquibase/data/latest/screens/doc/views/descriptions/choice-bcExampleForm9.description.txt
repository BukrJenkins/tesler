
  Для выбора значений служат поля типа checkbox, checkboxSql, dictionary, pickList
  <br/>
  <b>checkbox</b> - чекбокс
  <br/>
  <b>checkboxSql</b> - чекбокс, который идентичен типу checkbox, но должен использоваться с <b>SQL-сущностями</b> для возможности корректной фильтрации по этому полю в виджетах с типом список.
  <br/>
  <b>dictionary</b> - справочник
  <br/>
  <b>pickList</b> - пик-лист
  <br><br>
  Для корректной работы поля типа pickList под View должен быть создан виджет с типом PickListPopup.<br>
  PickListPopup-виджет должен быть основан на БК, которая отбирает записи для выбора.<br>
  В поле WIDGET.FIELDS задаются поля, которые будут выведены в PickListPopup-виджет.
