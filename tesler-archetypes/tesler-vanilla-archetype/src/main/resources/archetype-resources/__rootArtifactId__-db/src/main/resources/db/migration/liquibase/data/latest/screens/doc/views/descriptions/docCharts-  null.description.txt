
<h3>График bubble</h3>
<br>
Для создания диаграммы ганта необходимо в блоке chart указать engine: 'highcharts', а в блоке chart.meta.type указать bubble
<br>
Данные для построения графика указываеются в блоке chart.series.data, если параметр заканчивается на Key, то данные берутся из запроса, иначе берутся из самого параметра
<br>
Обязательные параметры в data это y|yKey и z|zKey, а так же, если тип оси X category, то name|nameKey, иначе x|xKey
<br>
Все оставшиеся используемы параметры для работы описаны на <a href="https://api.highcharts.com/highcharts">странице с официальной документацией</a>
