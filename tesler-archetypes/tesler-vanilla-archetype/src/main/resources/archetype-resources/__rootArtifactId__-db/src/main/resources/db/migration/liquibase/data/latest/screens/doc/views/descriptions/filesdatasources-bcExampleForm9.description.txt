
  По умолчанию загрузка файлов происходит в табличку file_entity. Есть возможность кастомизировать таблицу-источник:
  Данные по источникам хранятся в табличке FILE_DATASOURCE (Название таблички и колонок с данными по файлу)
  Для возможности загрузки файлов(upload) в FILE_DATASOURCE необходимо указать все колонки.
  Для возможности закачки файлов(download) в FILE_DATASOURCE можно не указывать колонки с MIME-типом и размером файла
  <br/>
  <br/>
  <b>fileSource</b> - поле с названием источника файла
