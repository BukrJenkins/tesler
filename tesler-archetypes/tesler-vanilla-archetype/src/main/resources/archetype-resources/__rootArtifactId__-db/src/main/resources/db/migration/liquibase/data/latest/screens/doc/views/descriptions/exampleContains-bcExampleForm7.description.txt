В системе присутствуют возможность поиска записи по значению в поле. Для этого необходимо указать, что это по этому полю разрешен поиск в FieldMetaBuilder,
 а также указать в DTO аннотацию @SearchParameter у поля по которому происходит поиск. В данном примере реализован поиск с условием "содержит" (contains)
<br>
<br>

<b>Доступные типы данных для фильтра contains:</b><br>
- input<br>
- text<br>
- fileUpload<br>
