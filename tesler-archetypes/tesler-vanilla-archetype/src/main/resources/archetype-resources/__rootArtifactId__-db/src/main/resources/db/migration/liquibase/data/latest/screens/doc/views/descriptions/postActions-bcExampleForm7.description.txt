<p>
	Пост-экшны выполняют каку-либо стандартную процедуру после выполнения некоторого основного, чаще всего инициированного пользователем действия.
</p>

Различают следующие пост-функции:
<ul>
    <li>выполнить обновление БК;</li>
    <li>скачать файл;</li>
    <li>скачать файл по ссылке;</li>
    <li>открыть пиклист;</li>
    <li>выполнить дриллдаун (внутри приложения или по относительным\абсолютным ссылкам) с возможностью выбора в текущей или новой вкладке;</li>
    <li>выполнить отложенное обновление БК;</li>
    <li>отображение сообщения пользователю (ошибка, предупреждение, уведомление)</li>
</ul>
<p>
Разновидности пост-функций можно детально изучить в классе PostAction бэк-проекта.<br />
Пример прикрепления пост-функций можно увидеть в VanillaDocServiceImpl.
</p>
