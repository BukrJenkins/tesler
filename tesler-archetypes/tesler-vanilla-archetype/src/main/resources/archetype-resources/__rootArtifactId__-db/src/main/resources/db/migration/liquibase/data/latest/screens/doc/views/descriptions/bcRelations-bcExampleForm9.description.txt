При изменении активной записи в List-виджете, система поддерживает каскадное обновление данных в дочерних виджетах.<br>
Связь данных на разных виджетах осуществляется за счет связанных БК, на котором основаны виджеты.<br>
Для связи данных виджетов необходимо для дочернего виджета использовать БК, которая имеет связь на БК родительского виджета по полю BC.PARENT_NAME
