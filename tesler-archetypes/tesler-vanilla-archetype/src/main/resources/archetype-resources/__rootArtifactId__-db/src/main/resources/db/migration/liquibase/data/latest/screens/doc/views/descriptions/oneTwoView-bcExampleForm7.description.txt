Элементы располагаются в одну и две колонки. Для такого расположения используется шаблон DashboardView. В таблицу ViewWidgets значение GRID_WIDTH ставится равным 1 для элементов, которые будут занимать половину экрана, и 2 для элементов, которые будут занимать всю ширину. Элемент с GRID_WIDTH = 1 будет занимать всю ширину, если он находится между элементами с GRID_WIDTH = 2.
