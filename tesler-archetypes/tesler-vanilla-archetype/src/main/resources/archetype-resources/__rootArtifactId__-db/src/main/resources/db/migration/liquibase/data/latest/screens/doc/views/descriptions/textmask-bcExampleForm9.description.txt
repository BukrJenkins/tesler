
 Для вывода текста служат поля типа text и input
 <br/>
  <b>text</b> - многострочный текст
  <br/>
  <b>input</b> - однострочный текст
  <br/>
  <b>richText</b> - многострочный текст с форматированием (HTML)
  <br/><i>Для начала редактирования - клик в область с текстом или на кнопку "Редактировать"</i>
  <br/><i>Для завершения редактирования - кнопка "Завершить редактирование", <b>Внимание!</b> если редактирование не завершено внесенные в текст изменения не будут сохранены</i>
