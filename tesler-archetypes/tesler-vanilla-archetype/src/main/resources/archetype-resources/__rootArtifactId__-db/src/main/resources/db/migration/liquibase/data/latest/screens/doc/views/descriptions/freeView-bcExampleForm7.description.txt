Элементы располагаются в CustomView. В данном режиме появляется специальная кнопка, которая позволяет перейти в режим редактирования виджетов, в котором можно самому выбрать ширину и высоту виджетов.Выбранное значения для виджетов хранится локально. Задается выставление значением в поле EDITABLE в 1 для любого шаблона
