<p>Сводная таблица на которую так же накладываются правила по группировке элементов <i>("сворачивания выборки" по осям <b>X </b>и Y)</i></p>
    <h3 style="z-index:auto;"><b>требования к конфигурации (таблица WIDGET) - <span style="color:#006400;">обязательно</span>:</b></h3>
                <ul>
                	<li><b>FIELDS:</b> необходимо описать минимум три поля, которые будут соответственно представлять "Колонку", "Строку" и "Ячейку" (значение - которое как раз будте представлено сгруппированно в отношении к осям), для этого поля снабжаются соответствующими свойствами булевого значения: isRow, isCol, isValue (см. пример SQL-INSERT)</li>
                	<li><b>AXIS_FIELDS:</b> специфическая группа параметров, предназначенная именно для двумерных фильтров, представляет собой список достпуных для выбора осей, среди которых обязательно должны присутствовать установленные в FIELDS как "Строка" и "Колонка" оси, их необходимо отметить соответствующими булевыми свойствами xDefault и yDefault (отражающими установку значения для этих осей "по умолчанию") данные свойства можно комбенировать с булевым свойством fixedAxis что позволит заблокировать знеачение для соответствующей оси для дальнейшего редактирования со стороны пользователя (см. ось "X" на интерактивном примере)</li>
                </ul>
    
        <h3 style="z-index:auto;"><b>требования к конфигурации (таблица BC_PROPERTIES) - <span style="color:#008080;">опционально </span>:</b></h3>
        <p>для двумерных фильтров так же предусматривается возможность установить набор штатных преднастроенных фильтров, для этого надо добавить соответствующий <i>JSON </i>который опишет доступные фильтры для соответствующего двумерного фильтра <i>(привязка по имени БК)</i>, каждый фильтр описывается <i>JSON-нодой</i> c 2-мя свойствами, где <b>name </b>- наименование преднастроенного фильтра отображаемого пользователю при настройке осей в соотв.выпадающем списке <i>(если фильтры для БК не определены - этот список не отображается вовсе)</i>, а <b>value </b>- это фактически URI-параметр который будет добавлен к запросу для соотв.БК</p>
    
        <h3 style="z-index:auto;">управление:</h3>
        <p>Для управления настройками двумерного фильтра требуется раскрыть пункт меню виджета <i>"Дополнительно" </i>(шестеренка) в выпадающем списке - пункт <i>"Задать оси"</i></p>
