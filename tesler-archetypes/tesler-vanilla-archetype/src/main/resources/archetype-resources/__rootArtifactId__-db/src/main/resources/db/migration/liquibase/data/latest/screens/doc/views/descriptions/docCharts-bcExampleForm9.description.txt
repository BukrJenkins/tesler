
<h3>Графики</h3>
<br>
Для работы с графиками используется колонка <b>WIDGET.CHART</b>, которая содержит json-массив объектов, где каждый объект описывает отдельный график, между которыми можно переключаться на виджете.
<br>
<br>
<b>Ручное форматирование числовых значений при экспорте/выводе на печать</b>
<br>
Ручная настройка отображения значений производится через задание шаблона вывода в поле <b>meta.exporting.chartOptions.series.dataLabels.format</b> json-объекта настроек графика.
<br>
Шаблон стандартного для Highcharts вида: <b>"{point.y:.3f}"</b>, где 3 - количество знаков после запятой.
