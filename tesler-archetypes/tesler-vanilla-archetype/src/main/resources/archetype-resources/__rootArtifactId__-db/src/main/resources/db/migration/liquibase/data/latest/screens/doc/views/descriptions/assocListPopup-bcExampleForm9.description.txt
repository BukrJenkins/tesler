Виджеты AssocListPopup используются при создании связей 'Многие ко многим'.<br><br>
Для правильной работы должны соблюдаться следующие правила:
<ul>
<li>  1) Название БК на AssocListPopup-виджете должно быть идентично БК на List-виджете, но с добавлением 'Assoc' в конце;</li>
Например:<br>
- bcExampleRelatedRecords: БК List-виджета<br>
- bcExampleRelatedRecordsAssoc: БК AssocListPopup-виджета<br>
<li>  2) При написании БК для AssocListPopup-виджета необходимо делать фильтрацию, которая не допускает отображение на этом виджете уже добавленных записей.</li>
</ul>
