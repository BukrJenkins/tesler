<p>
	Пре-экшны отображают модальные окна-предупреждения перед выполнением экшена.
</p>
<br/>
<br/>
Различают следующие виды предупреждений:
<br/>
<br/>
<ul>
    <li>- Подтверждение выполнения с возможностью отмены действия (<b>confirm</b>); Пример: кнопка 'Подтверждение'</li>
    <li>- Предупреждение о выполняемом действии (<b>info</b>); Пример: кнопка 'Предупреждение'</li>
    <li>- Сообщение о недоступности действия (<b>error</b>); Пример: кнопка 'Ошибка'</li>
</ul>
<br/>
Пре-экшен добавляется при добавлении действия, при этом можно задать кастомное сообщение для вывода пользователю.
