Существует два вида виджетов с типом HeaderWidget:
<ul>
<li> - HeaderWidget со статическим текстом (Список контрагентов)</li>
<li> - HeaderWidget с динамическим текстом (Контрагент: ${legalPersonName})</li>
</ul><br>
HeaderWidget при создании может быть привязан к компоненте (БК).<br>
Для вывода статического текста в HeaderWidget, необходимо заполнить поле WIDGET.TITLE<br>
Для создания HeaderWidget с динамическим текстом необходимо связать виджет с БК родительского виджета, в поле WIDGET.TITLE указать динамический параметр в формате:<br>
${bcField}<br>
где bcField - поле БК, к которой привязан виджет.<br><br>
При изменении активной записи на родительском List-виджете, основанном на той же БК, что и HeaderWidget, значение bcField будет динамически меняться.<br><br>

В поле OPTIONS таблицы WIDGET устанавливаются общие опциональные настройки виджета в формате json-объекта.<br>
Для установки цвета фона Header-виджета, необходимо в json-объект колонки OPTIONS добавить поле "bgColorKey", указав название поля виджета из DTO, в котором передается цвет.<br><br>
Пример:<br>
<pre><b>
{
    "bgColorKey": "intendModeBgColor"
}</b></pre>
<br>
Следует учитывать, что при использовании поля bgColorKey на Header-виджете выставляется черный шрифт и белый фон, и только после этого применяется цвет, переданный в указанном поле DTO.
