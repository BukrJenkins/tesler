
При создания связей 'Многие ко многим' используются виджеты <b>AssocListPopup</b>.<br/>
Для работы требуются два виджета с типами:<br/>
<b>- List</b>: содержит список связанных записей. При попытке добавить запись вызывает AssocListPopup.<br/>
<b>- AssocListPopup</b>: всплывающий виджет, содержит список записей, которые можно добавить на List-виджет.<br/>
<br/>
<b>Важно:</b><br/>
Для правильной работы необходимо, чтобы название БК на AssocListPopup-виджете было идентично БК на List-виджете, но помимо этого содержало в конце 'Assoc'.<br/>
Например:<br/>
- <b>bcExampleRelatedRecords</b>: БК List-виджета<br/>
- <b>bcExampleRelatedRecordsAssoc</b>: БК AssocListPopup-виджета<br/>
<br/>
При написании БК для AssocListPopup-виджета необходимо делать фильтрацию, которая не допускает отображение на этом виджете уже добавленных записей.
