
<h3>Таблица, отображающая набор записей заданной сущности.</h3>
<br>
<h3>Настройка списка. Таблица WIDGET:</h3>
<p>Основные поля:</p>
<p>- <b>ID:</b> уникальный идентификатор виджета</p>
<p>- <b>TYPE:</b> <b>"List"</b></p>
<p>- <b>TITLE:</b> Заголовок, отображаемый на таблице.</p>
<p>- <b>BC:</b> Код бизнес-компоненты, которая будет использоваться для выборки данных.</p>
<p>- <b>FIELDS:</b> <u>JSON-массив</u>, содержащий объекты, которые описывают колонки таблицы и соответствующие им поля на выбираемых записях.</p>
<br>
<pre><b>WIDGET.FIELDS[i]</b></pre>
<p>Основные поля описания колонки:</p>
<p>- <b>title:</b> Выводимое название колонки</p>
<p>- <b>key:</b> Код поля записи, которому соответствует поле колонки</p>
<p>- <b>type:</b> Тип, определяющий способ отображения данных в колонке</p>
<p>- <b>hintKey:</b> Ключ поля БК содержащего расшифровку или пояснение для сокращенного значения (целевое поле обязательно строковое)</p>
<p>- <b>width:</b> Ширина колонки по-умолчанию (число)</p>
<br>
<pre><b>WIDGET.OPTIONS[i]</b></pre>
<p>Дополнительные опции:</p>
<p>- <b>multilineColumnHeaders:</b> (true/false) - определяет необходимость отображения заголовков таблицы в многостроковом виде с переносом "по словам"</p>
<p>- <i>см.пример в виджете "Таблица"</i></p>
<p>- <b>multiline:</b> (true/false) - определяет необходимость отображения ячеек таблицы в многостроковом виде с переносом "по словам"</p>
<br>
<pre><b>WIDGET.FIELDS[i].type</b></pre>
<p>Доступные типы колонок:</p>
<p>- <b>number</b></p>
<p>- <b>input</b></p>
<p>- <b>date</b></p>
<p>- <b>dateTime</b></p>
<p>- <b>dateTimeWithSeconds</b></p>
<p>- <b>checkbox</b></p>
<p>- <b>checkboxSql</b> (для таблиц на основе SQL-сущностей)</p>
<p>- <b>DMN</b></p>
<p>- <b>pickList</b></p>
<p>- <b>dictionary</b></p>
<p>- <b>hidden</b></p>
<p>- <b>text</b></p>
<p>- <b>percent</b></p>
<p>- <b>fileUpload</b></p>
<p>- <b>money</b></p>
<p>- <b>combo-condition</b></p>
