Элементы располагаются в несколько колонок в зависимости от параметра GRID_WIDTH. Ширина экрана занимает 4 значения GRID_WIDTH. Элементы располагаются последовательно в зависимости от параметра POSITION и занимают столько, сколько указано в GRID_WIDTH.Для такого расположения используется шаблон QuadColumnView.
