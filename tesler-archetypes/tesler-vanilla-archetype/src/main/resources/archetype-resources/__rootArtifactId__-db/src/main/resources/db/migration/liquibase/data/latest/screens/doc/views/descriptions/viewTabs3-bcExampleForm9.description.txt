Для создания переключателя View (ViewTab) необходимо завести View всех уровней вложенности в таблице VIEW и привязать их к экрану через таблицу SCREEN_VIEWS.<br>
В таблице SCREEN настроить навигацию по созданным View, указав их в поле NAVIGATION в структуре json. Список доступных для экрана View должен быть задан в блоке child[] в параметре viewId - идентификатор View из таблицы VIEW (см. фрагмент кода ТАБ 1).<br><br>
Для добавления на экран переключателя View необходимо создать виджет с типом SecondLevelMenu, ThirdLevelMenu или FourthLevelMenu.
