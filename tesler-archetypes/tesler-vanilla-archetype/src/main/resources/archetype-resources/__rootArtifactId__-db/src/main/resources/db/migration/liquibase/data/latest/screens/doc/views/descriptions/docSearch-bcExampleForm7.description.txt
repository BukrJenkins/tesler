В системе присутствуют возможность поиска записи по значению в поле. Для этого необходимо указать, что это по этому полю разрешен поиск в FieldMetaBuilder,
 а также указать в DTO аннотацию @SearchParameter у поля по которому происходит поиск.

<h3><b>Виджет-фильтры:</b></h3><br>
Имеется возможность отображать необходимые фильтры на конкретном виджете сразу после загрузки view.
 На данный момент, такие фильтры нельзя удалить с панели фильтров виджета.<br>
Для добавления таких фильтров необходимо в колонке виджета <b>OPTIONS</b> добавить на JSON-объект поле showFilters, которое представляет собой
 массив объектов, описывающих пустые фильтры виджета.<br>
Формат объектов:
<pre>{
  "key": "inn",
  "type": "input",
  "label": "ИНН"
}</pre>
<br>

<b>key</b> - Ключ фильтра.<br>
<b>type</b> - Тип фильтра.<br>
<b>label</b> - Заголовок фильтра, который будет выведен в случае, если не найдено название фильтруемого поля (например,
 если данное поле не выведено на виджет). Необязательное поле.
