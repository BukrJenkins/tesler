В системе присутствуют ошибки трех видов: по полям, бизнес-ошибки и системные. Ошибки по полям - это ошибки вида "незаполненное поле".
Бизнес ошибки - это ошибки бизнес логики, которые обрабатываются на стороне сервера и вызываются исключением  businessException.
Системные ошибки - это ошибки связанные с работой программы. Например, ответ с NullPointerException отобразит системную ошибку
