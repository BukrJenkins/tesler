
 Для отображения подсказки используется поле hintText и hintTitle.
 <br/>
  <b>- hintText</b> - текст подсказки
  <br/>
  <b>- hintTitle</b> - заголовок подсказки. Отображается только при заполненном hintText.
  <br/>
  <b>- hintKey</b> - ключ другого поля БК которое содержитт соотносящееся значение пояснения/расшифровки для демонстации в тултипе при наведении на отдельно взятую ячейку колонки.
  <br/>
  <font color="green"><i><b>примеры для сводных таблиц см. на странице документации "Виджеты" => "Сводная таблица"</b></i></font>
  <br/>
  <b>см. ячейки в <i>"Сводная таблица (Устаревшая..."</i> и <i>"Сводная таблица с групировкой по строкам и дополнительным столбцом"</i></b>
