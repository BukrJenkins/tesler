В системе присутствуют возможность поиска записи по значению в поле. Для этого необходимо указать, что это по этому полю разрешен поиск в FieldMetaBuilder,
 а также указать в DTO аннотацию @SearchParametr у поля по которому происходит поиск. В данном примере реализован поиск с условием "содержит одно из" (containsOneOf)
<br>
<br>

<b>Доступные типы данных для фильтра containsOneOf:</b><br>
- input<br>
- fileUpload<br>
